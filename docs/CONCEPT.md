## How It Works

### 1. Transaction Fees

Every transfer applies a **configurable fee** (default 6%):

```
Example on 1000 token transfer:
- 60 tokens collected as fee
- 940 tokens transferred to recipient
```

### 2. Fee Allocation

Fees are split deterministically:

- **40%** → Index Rewards Pool (holder rewards)
- **30%** → Buyback Pool (market support)
- **20%** → Liquidity Reinforcement Pool
- **10%** → Burn (deflationary pressure)

### 3. Flywheel Triggers

When thresholds are reached:

1. **Buyback Execution**
   - When buyback pool ≥ threshold
   - Tokens bought from market
   - Creates upward pressure

2. **Liquidity Addition**
   - When liquidity pool ≥ threshold
   - LP tokens added to DEX
   - Strengthens market structure

3. **Reward Distribution**
   - Proportional to holdings
   - Claimable by holders
   - Incentivizes holding

### 4. Index Rebalance Cycle

The system rebalances periodically:

```
Cycle N begins
  ↓
Fees accumulate in pools
  ↓
Volume increases
  ↓
Thresholds reached
  ↓
Execute: Buybacks + Liquidity + Rewards
  ↓
Record cycle metrics
  ↓
Cycle N+1 begins
```

## Snowball Effect

As volume increases:
- Pools fill faster
- Rebalances occur more frequently
- Rewards compound
- Liquidity deepens
- Market structure strengthens

This creates a **positive feedback loop** - the flywheel or "snowball" effect.
